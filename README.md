# Movie Website 🎬

This is a movie website created using a no-code platform (Lovable).  
It showcases movie details with a clean and visually appealing interface.

## 🚀 Features
- Modern UI design
- Movie listing display
- Easy navigation

## 🛠️ Tools Used
- Lovable (No-code platform)
- Supabase for managing database

## 🔗 Live Project
https://movieshow.lovable.app
